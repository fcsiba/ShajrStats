# Shajr Stats
![logo](app/static/img/shajrstats.png)

Scans satelite imagery to generate data drive dashboards regarding the vegetation greenery of Pakistan
On a mission to *#MakePakistanGreenAgain*

# Usage
`source venv/bin/activate`
`flask run app`

## Dependencies
- Flask
- Pandas
- Gunicorn
